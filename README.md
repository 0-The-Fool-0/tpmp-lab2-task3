# tpmp-lab2-task3

## Overview
Программа для работы со структурой STUDENT, содержащей информацию о студентах:
- Фамилия и инициалы
- Курс
- Успеваемость (5 оценок)

Программа выполняет:
1. Ввод данных о 10 студентах
2. Сортировку по алфавиту
3. Вывод отсортированного списка
4. Вычисление общего среднего балла
5. Вывод студентов, чей средний балл выше общего
